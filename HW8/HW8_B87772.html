The formula of dynamic time warping algorithm is: For the values on left in matrice: │Ai - Bj │+ D [i-1,0]

For the elements on bottom of matrice: │Ai - Bj │+ D [0, j-1]

For the remaining elements: │Ai - Bj │+ min (D [i-1, j-1] D [i-1, j] D [i, j-1])

EX2: Extendible hashing is a dynamic hashing scheme. The difference between Dynamic and Static Hashing is that there is no need to recalculate new bucket address and touch items in all the other buckets in the hash table in Dynamic Hashing.

The dynamic hashing was invented to solve the problem of changing the size of the hash table. In static hashing, in order to change the hash table size, one must change the hashing function to one with a bigger table and re-hash all elements with the new function. I dynamic hashing, the table is split into buckets which hold multiple slots. When changing the size it is then only necessary to split the concerned buckets.

EX3: Extendable Hashing is designed to minimize the cost of rehashing. A Target bucket B will be split into B and B. when B is generated, B will be rehashed between B and B. Hash function gives a hash value. Hash values is stored in hash table which is stored in main memory. Overflowing of buckets is not acceptable here. In Extendible Hashing, an auxiliary data structure is used which is called bucket directory. The principle is that pointers are used each entry to the buckets in hash table. Bucket directory is necessary structure here. When overflow happens, the B bucket is splitted to B and B and bucket directory should be updated. Splitting is managed depending on global and local depth. Global Depth is equal to the number of bits used from the hash value generated by hash function. Local depth is calculated by buckets. While inserting if local depth is more than global depth, it is created new bucket to insert new value.

Example:

  1. hash(4) = 100
  2. hash(6) = 110
  3. hash(1) = 01
  4. hash(5) = 101

from random import randint
import sympy 
import matplotlib.pyplot as plt
import statistics
import hashlib
m=5000000                     
f = open('words.txt','r',encoding='utf-8')      
data={}                         
                                
p=sympy.nextprime(m)
def ord_hash(word):
    h=0
    for c in word:
        h+=ord(c)
    return h

def len_hash(word):
    return len(word)
def get_collision(t):
    collision=0
    for i in t:
        if i > 1:
            collision+=i-1
    return collision
table=[0 for i in range(0,m)]        
for line in f:
    table[ord_hash(str(line))]+=1
print("most used int: ", table.index(max(table)))
data['ord_hash']=get_collision(table)
f.seek(0)

table=[0 for i in range(0,m)]        
for line in f:
    table[len_hash(str(line))]+=1
print("most used int: ", table.index(max(table)))
data['len_hash']=get_collision(table)
f.seek(0)

table=[0 for i in range(0,m)]        
for line in f:
    line = line.encode('utf-8')
    table[int(hashlib.sha256(line).hexdigest(), 16) % m]+=1
print("most used int: ", table.index(max(table)))
data['sha256']=get_collision(table)
f.seek(0)
